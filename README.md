# Tags-Counter-Crawler
-----------------------
Script that will give you the information about tags inside any website in Java
--------------------------------------------------------------------------------
###Input : URL
###Output :
  * Total Number of tags.
  * Number of unique tags.
  * Frequency of each Tags.
